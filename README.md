# card-print-aligner-
this webapp is a card generation tool for printers that prints one face at a time, because i'm a person who hates going outside to a public library to get a copy of my personal cards i made this tool so i can print copies on my single faced printer without the humiliation ritual of leaving my house. 
the UI is in arabic to express my frustration about the haslte of getting paper copies of cards that already contains chips inside them THERE IS A CHIP INSIDE WHY DONT YOU JUST READ IT WITH THE PROPER TOOL SAVE THE DATA YOU NEED AND DO THE REST OF MY PAPER WORK WHATS THE POINT !!!!
Also i'd like to note that you're gonna need python and flask for the backend to run .
